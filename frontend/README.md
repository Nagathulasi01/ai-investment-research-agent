# AI Investment Agent

A simple AI-powered investment analysis web application built using React.js and Node.js.

## Features

* Analyze company investment potential
* Company summary generation
* Strength and risk analysis
* Investment recommendations
* Responsive user interface

## Tech Stack

### Frontend

* React.js
* CSS

### Backend

* Node.js
* Express.js

### AI Logic

* Rule-based AI-style investment analysis
* Backend prepared for Gemini API integration

---

## How to Run the Project

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
